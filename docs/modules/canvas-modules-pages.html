canvas:setup_modules (List)
Create/edit all modules.

Parameters:

  • List table of ordered module names to create. If names are different than the modules currently defined, new ones are created and/or current modules are re-ordered. If modules exist that aren't specified, the function will offer to delete them (case-by-case).
canvas:update_module (module_name, ask, module)
Create/edit contents of an individual module.

Parameters:

  • module_name string
  • ask string of whether to proceed — empty asks, or |"y"| does, or anything else does not
  • module table items (see Canvas API documentation for raw syntax) Table of module items has some shorthand definitions defined in the code. TODO: document these properly.
